# 版本V1.0.2改动
## 首页
1. **AI实验室 - 小智bot粘滞修复（核心问题）**
   - V1.0.1中"小智bot冻结在页面上方"的需求**未实质性解决**，需重新实现
   - 要求：小智bot必须**真正粘滞(sticky)在页面顶部**，滚动时始终可见
   - 下拉历史记录时，小智bot不被推走或遮挡
   - 大小幅度适当缩小，但不影响交互体验

2. **编程探险岛 - 网格对齐系统重构（核心问题）**
   - **小猫出生点**：必须在游戏UI网格的交叉点上，不能有像素偏移
   - **星星生成点**：每个星星的坐标必须严格落在网格点上
   - **移动步进**：每次前进/转向后的落点都必须精准落在网格点上（网格 snapping）
   - 本质是建立一套统一的网格坐标系，所有游戏元素（小猫、星星、障碍物）都基于此坐标系定位

3. **编程探险岛 - 指令堆栈可视化**
   - 在游戏界面中实时展示**最近N条已执行的指令记录**
   - 类似终端/控制台的效果：每执行一条指令就追加一行显示
   - 显示内容包括：指令类型（前进/左转/右转）、执行序号、执行时间
   - 目的：防止玩家忘记前进了多少步，提供操作回溯能力
